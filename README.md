# UnderCore
Projet Universitaire de première année de BTS CIEL

## - Comment ouvrir les fichiers sources ?
-> Télécharger la dernière release
-> Dézipper le fichier
-> Télécharger Unity Hub
-> Ouvrir via l'onglet "Add" un fichier sur votre disque
-> Selectionner le dossier UnderCore 1.0.0
-> Lorsque le projet est ouvert allez sur l'onglet 'Project' puis dans Asserts/Core/Scenes et double cliquer sur UnderCore (Scene Assets)
Et voici le fichier source complet !

## - Comment jouer ?
-> Télécharger la dernière release
-> Dézipper le fichier
-> Aller dans le dossier 'UnderCore-Build'
-> Double cliquer UnderCore.exe
Et bon jeu !

## - Commandes de jeu :
-> Déplacements :
  -> Haut : Z/↑
  -> Bas : S/↓
  -> Droite : D/→
  -> Gauche : Q/←
-> Interaction Général : F
-> Menu :  Echap
