# UnderCore
Projet académique de première année de BTS CIEL

## - Comment ouvrir les fichiers sources ?
- Télécharger la dernière release<br>
- Dézipper le fichier<br>
- Télécharger Unity Hub<br>
- Ouvrir via l'onglet "Add" un fichier sur votre disque<br>
- Selectionner le dossier UnderCore 1.0.0<br>
- Lorsque le projet est ouvert allez sur l'onglet 'Project' puis dans Asserts/Core/Scenes et double cliquer sur UnderCore (Scene Assets)

Et voici le fichier source complet !

## - Comment jouer ?
- Télécharger la dernière release<br>
- Dézipper le fichier<br>
- Aller dans le dossier 'UnderCore-Build'<br>
- Double cliquer UnderCore.exe

Et bon jeu !

## - Commandes de jeu :
- Déplacements :<br>
&ensp;- Haut : Z/↑<br>
&ensp;- Bas : S/↓<br>
&ensp;- Droite : D/→<br>
&ensp;- Gauche : Q/←<br>
- Interaction Général : F<br>
- Menu :  Echap
